HOW TOs

- Export .pt model as .mlpackage artifact for easy integration into mobile app:
python yolo/lazy.py task=export model=v9-c task.output_path=weights/v9-c-ios.mlpackage
python yolo/lazy.py task=export model=v9-c task.image_size=[384,640] task.output_path=weights/v9-c-ios-384x640.mlpackage   
  - Override config settings
python yolo/lazy.py task=export model=weights/sku-110-best 'task.image_size=[384,640]' task.output_path=weights/sku-110-384x640.mlpackage dataset.class_num=1 'dataset.class_list=[item]'  

- Train on SKU-110K dataset from smallest v9 model.
python yolo/lazy.py task=train dataset=sku110k model=v9-t task.epoch=10 task.data.batch_size=8 task.data.cpu_num=8 accelerator=mps device=1 weight=True name=v9t-sku110k use_wandb=False 2>&1

 - Resume model training for more epochs:
 python yolo/lazy.py task=train dataset=sku110k model=v9-t task.epoch=50 task.data.batch_size=8 task.data.cpu_num=8 accelerator=mps device=1 weight=True name=v9t-sku110k use_wandb=False '+resume="runs/train/v9t-sku110k/checkpoints/epoch=9-step=2371.ckpt"' 
 -For future resumes after this run, you'd use last.ckpt instead:                                                                                                               
                                                                                            
  '+resume="runs/train/v9t-sku110k/checkpoints/last.ckpt"' 

  ulimit -n 4096 && python yolo/lazy.py task=train dataset=sku110k model=v9-t task.epoch=50 task.data.batch_size=8 task.data.cpu_num=8 accelerator=mps device=1 weight=True name=v9t-sku110k use_wandb=True '+resume="runs/train/v9t-sku110k/checkpoints/epoch=9-step=2371.ckpt"'
